### 분석 과정
- 데이터 수집
  - 1998년~2014년동안 동계 올림픽이 개최된 해의 데이터 수집

      - GDP, 1인당 GDP, GNI, 1인당 GNI, 수출량

      - 국가면적, 위도(날씨 반영)
    - 기타 요인
      - 참가선수 수, 평균 경기 실적, 주최국 효과, 기대 수명
